### 專案與本次範圍摘要

本專案希望將校園社團的活動與場地申請流程數位化，降低紙本簽核延遲、時段衝突與狀態不透明。本次可操作切片聚焦「社團幹部建立活動、申請場地、檢查衝突並查看待審核狀態」。

### 本次納入

1. 活動申請表單與必填檢查。
2. 場地、日期及起訖時間選擇。
3. 同場地重疊時段的衝突檢查。
4. 無衝突時建立假資料申請並顯示「待審核」。
5. 申請紀錄與狀態顯示。

### 本次不納入

正式登入、正式資料庫、金流、LINE 或 Email 通知、自動推薦替代時段、跨校區整合、AI 自動排程及完整審核後台。
