#### 1. 專案背景

```text
本專案為「校園社團活動與場地智慧管理平台」，希望改善目前紙本活動申請、場地借用及活動報名流程效率低落的問題。透過數位化方式，讓社團幹部、課外活動組老師及學生能更快速完成活動申請、場地管理與活動報名，提升行政效率並降低場地衝突。
```

#### 2. 目標角色

|      角色      | 主要任務           | 主要痛點                         |
| :------------: | ------------------ | -------------------------------- |
|    社團幹部    | 建立活動、申請場地 | 紙本申請流程繁瑣、審核時間長     |
| 課外活動組老師 | 審核活動與場地     | 人工確認場地衝突耗時             |
|    一般學生    | 查看活動並報名     | 不容易確認是否報名成功及剩餘名額 |

#### 3. 第一個可操作切片（First Working Slice）

```text
建立「社團幹部新增活動並申請場地」功能，完成後系統可顯示活動資訊，並檢查場地時段是否衝突，若無衝突則送出申請並顯示「審核中」狀態。
操作流程：

1. 社團幹部進入活動場地申請頁面。
2. 填寫活動名稱、日期、開始與結束時間、場地、預估人數及活動說明。
3. 系統檢查必填欄位與時間格式。
4. 系統檢查同一場地在所選時間是否已有重疊申請。
5. 無衝突時建立申請，初始狀態為「待審核」；有衝突時阻止送出並顯示原因。
6. 社團幹部可在申請紀錄中查看送出結果與目前狀態。
```

#### 4. 本期範圍內項目（In-Scope）

1. 活動建立
2. 場地申請
3. 場地衝突檢查
4. 活動列表顯示
5. 審核狀態顯示（審核中）
6. 必填欄位及日期、時間格式

#### 5. 本期範圍外項目（Out-of-Scope）

1. 正式登入系統
2. 正式資料庫
3. 金流付款功能
4. LINE 通知
5. 自動排程演算法
6. 跨校區場地整合
